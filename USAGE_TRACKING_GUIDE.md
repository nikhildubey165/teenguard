## How It Works

### For Teenagers

#### 1. Track App Usage
- Go to **"Track Usage"** tab in your dashboard
- You'll see all available apps (YouTube, Instagram, Games, etc.)

#### 2. Start a Timer
- Click **"‚ñ∂ Start Timer"** on any app you want to use
- The timer starts counting automatically
- You can see:
  - Today's total usage (including the running timer)
  - Your daily limit (if set by parent)
  - Progress bar showing how close you are to the limit

#### 3. Stop the Timer
- When you're done using the app, click **"‚èπ Stop Timer"**
- The time is automatically saved to your usage record
- You'll see a confirmation message

#### 4. Manual Entry
- Click **"+ Manual Entry"** button
- Select an app
- Enter the number of minutes you used
- Click **"Add Usage"**
- Useful if you forgot to start the timer

#### 5. View Active Timers
- At the bottom, you'll see all currently running timers
- You can stop any timer from there

### For Parents

#### 1. View Usage Reports
- Go to **"Reports"** tab in your dashboard
- Select a teenager (or "All Teenagers")
- Choose time period (7, 14, or 30 days)

#### 2. What You'll See
- **Summary Cards**: Total usage per app, averages, days used
- **Detailed Table**: Daily breakdown showing:
  - Date
  - App name
  - Minutes used
  - Daily limit
  - Status (Within Limit / Over Limit)
  - Which teenager

#### 3. Real-Time Updates
- Reports auto-refresh every 30 seconds
- Click **"üîÑ Refresh"** button for immediate update
- See usage as it happens!

#### 4. Identify Issues
- Red highlighting = Over limit
- Green = Within limit
- Gray = No limit set

## Tips

### For Teenagers:
1. **Start Timer Before Using**: Always start the timer before opening an app
2. **Stop When Done**: Don't forget to stop the timer when you're finished
3. **Check Limits**: Check your limits before starting to avoid going over
4. **Manual Entry**: Use manual entry if you forgot to start the timer

### For Parents:
1. **Set Limits First**: Set app limits before teenagers start tracking
2. **Check Reports Regularly**: Monitor usage in the Reports tab
3. **Filter by Teenager**: Use the dropdown to see individual teenager usage
4. **Use Time Periods**: Switch between 7/14/30 days to see trends

## Troubleshooting

### Timer Not Saving
- Make sure you're logged in
- Check internet connection
- Try stopping and starting again

### Usage Not Showing in Reports
- Reports update every 30 seconds
- Click "Refresh" button
- Make sure teenager has tracked usage

### Can't Start Timer
- Check if daily limit is reached
- Make sure you're logged in as teenager
- Refresh the page

## Notes

- **Timers are saved locally**: If you close the browser, timers will resume when you come back
- **Time is tracked in minutes**: The system rounds to the nearest minute
- **Daily limits reset**: Limits reset at midnight
- **Multiple apps**: You can track multiple apps at the same time
